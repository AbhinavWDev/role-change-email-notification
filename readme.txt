== Description ==

The "Dynamic Role Change Email Notification" plugin allows you to send email notifications when a user's role is changed on your WordPress website. You can also customize the email templates with dynamic content.

Key Features:

- Send email notifications when a user's role changes.
- Customize the email subject, message, button style, and button text.
- Use placeholders in the email message to display dynamic content.

== Installation ==

1. Upload the "dynamic-role-change-email-notification" folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the email settings under the 'Email Settings' menu in the WordPress admin.

== Usage ==

1. After activating the plugin, go to the 'Email Settings' menu in the WordPress admin.
2. Customize the email subject, message, button style, and button text as per your requirements.
3. Save your settings.
4. When a user's role is changed, they will receive an email notification with the customized content.

== Frequently Asked Questions ==

= Can I customize the email templates for different user roles? =

Currently, this plugin allows you to customize the email templates globally. However, you can use placeholders in the email message to display dynamic content, such as the new user role.
